## How to Download

### Option 1: Download Everything (Easiest)

**On this page:** Click the green **`Code`** button в†’ **`Download ZIP`**

This downloads everything as a single ZIP file. Extract it and you have all the PDFs.

### Option 2: Clone with Git

If you know Git:

```bash
git clone https://github.com/Mumbai-University-PYQ/PYQ.git
cd PYQ
# You now have all PDFs organized by course
```

### Option 3: Download Single Course (Without Git) 

1. Go to folder: `BSc-IT/` (or your course)
2. Click that folder
3. Click green `Code` button в†’ `Download ZIP`
4. Extract

Done. All papers for that course are now local.
